What is Ghertist 3D Chatroom?

Free to use Browser based Experimental 3D / VR Chatroom platform supporting audio chat, Talk to others using voicechat in 3D enviroments running in browser or Mobile

  • Powered by AFrame, Networked AFrame and A-Frame Enviroment component
  • Chat with others using your microphone
  • Make sure you have headphones on (otherwise it can create feedback for others)
  • 2 3D Enviroments to explore - more in pipeline
  • Move around with your avatar

Possible Future Features

  • Moderation
  • Chat Interface
  • More advanced 3D Enviroments
